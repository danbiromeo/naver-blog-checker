# PRD: 네이버 블로그 검사기

## 1. 목적

특정 네이버 블로그의 전체 글을 크롤링하여 포스팅 현황과 네이버 검색 노출 여부를 한눈에 파악한다.

---

## 2. 입력

- 네이버 블로그 주소 **여러 개** 입력 가능 (한 줄에 하나씩)
```
https://blog.naver.com/blogid1
https://blog.naver.com/blogid2
https://blog.naver.com/blogid3
```
- 글 주소를 넣어도 자동으로 블로그 주소로 변환
  - 예: `blog.naver.com/blogid/224245623058` → `blog.naver.com/blogid`
- 검사 기간 선택: **최근 15일 / 최근 1달 / 최근 3달 / 전체**
- 각 블로그별로 개별 검사 후 결과 표시

---

## 3. 기능

### 3-1. 블로그 글 전체 크롤링

- 블로그의 모든 글 수집
- 수집 항목: 글 제목, 작성일, 글 URL
- 총 글 수 표시

### 3-2. 포스팅 빈도 + 노출률 연계 분석

**기본 지표:**
- 월별 포스팅 수 (막대 차트)
- 평균 빈도: 주당, 월당
- 최근 30일 포스팅 수

**빈도-노출률 상관관계:**
- 주간별 포스팅 수 / 노출 / 미노출 / 노출률 테이블
- 산점도 차트: X축 = 주간 포스팅 빈도, Y축 = 노출률
- 빈도 구간별 평균 노출률 요약 (예: 주 1~2회 → 85%, 주 3~4회 → 52%)
- 여러 블로그 입력 시 블로그별 비교 가능

### 3-3. 노출 검사

네이버에서 각 글 제목을 따옴표(`"제목"`) 검색하여 노출 여부 판정.

**검색 URL 형식:**
```
https://search.naver.com/search.naver?ssc=tab.nx.all&where=nexearch&query={URL인코딩된 "제목"}&sm=tab_dgs&qdt=1
```

**판정 기준:**
- 검색 결과 HTML에 해당 블로그 ID(예: `hhncpfajcy750`)가 포함 → **노출**
- 포함되지 않음 → **미노출**

**속도 제한:**
- 요청 간 1~2초 딜레이 (네이버 차단 방지)
- 진행률 프로그레스 바 표시

## 4. 화면 구성 (Streamlit)

```
[블로그 주소 입력 (여러 줄)]  [검사 시작 버튼]

── 블로그 선택 탭 ──────────────────
[blogid1] [blogid2] [blogid3]

── 요약 (선택된 블로그) ────────────────
총 글 수: 152개
노출: 87개 | 미노출: 65개 (노출률 57.2%)
평균 빈도: 주 3.2회 | 월 13.8회
최근 30일: 12개

── 월별 포스팅 차트 ─────────────────
(막대 그래프)

── 글 목록 ─────────────────────────
제목 | 작성일 | 노출여부 | URL
필터: [전체] [노출만] [미노출만]

── 하단 ───────────────────────────
[CSV 다운로드] (선택된 블로그 또는 전체)
```

---

## 5. 결과 내보내기

CSV 다운로드 버튼 제공.

| 열 | 내용 |
|----|------|
| 블로그ID | 블로그 아이디 |
| 제목 | 글 제목 |
| 작성일 | YYYY-MM-DD |
| 노출여부 | 노출 / 미노출 |
| URL | 글 링크 |

## 7. 제약사항

| 항목 | 내용 | 대응 |
|------|------|------|
| 네이버 차단 | 빠른 연속 요청 시 IP 차단 가능 | 요청 간 1~2초 딜레이 |
| 검사 시간 | 글 100개 기준 약 3~5분 | 프로그레스 바로 진행률 표시 |
| 네이버 HTML 변경 | 검색 결과 구조 변경 가능 | 블로그 ID 문자열 매칭으로 단순화 |
